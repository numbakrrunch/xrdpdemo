XRD Provisioning Demo

Demo implementation of an XRD Provisioning service.

Services available on this host:

  • /xrdp?q={uri} - XRD Provisioning endpoint
  • /webfinger?q={uri} - Webfinger endpoint
These services are also described in /.well-known/host-meta

Here are some sample curl commands for manually interacting with the service:

  • POST - Add a new link (will not work if this exact link already exists): 
    curl -v -d "<Link rel='http://oexchange.org/spec/0.8/rel/user-target' type='application/xrd+xml' href='http://www.example.com/linkeater/oexchange.xrd' />" http://xrdpdemo.appspot.com/xrdp?acct=charlie%40ecece.com
  • PUT - Update that link: 
    curl -v -d "<Link rel='http://oexchange.org/spec/0.8/rel/user-target' type='application/xrd+xml' href='updated' />" "http://xrdpdemo.appspot.com/xrdp?acct=charlie%40ecece.com&rel=http%3A%2F%2Foexchange.org%2Fspec%2F0.8%2Frel%2Fuser-target&type=application%2Fxrd%2Bxml&href=http%3A%2F%2Fwww.example.com%2Flinkeater%2Foexchange.xrd" -X PUT
  • DELETE - Remove that link: 
    curl -v "http://xrdpdemo.appspot.com/xrdp?acct=charlie%40ecece.com&rel=http%3A%2F%2Foexchange.org%2Fspec%2F0.8%2Frel%2Fuser-target&type=application%2Fxrd%2Bxml&href=updated" -X DELETE
